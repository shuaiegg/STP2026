## Technical Lessons Learned (P0 Fixes)
- **React State Deadlock**: Avoid using strict `!form.keywords` to disable buttons when using automated browser testing, as simulated inputs may not trigger React's synthetic `onChange` events properly. Fixed by moving validation inside the handler.
- **OpenClaw Security Tripwire**: When `agent-browser` fetches excessively large DOM snapshots (`<<<EXTERNAL_UNTRUSTED_CONTENT>>>`), the Gateway may trigger a prompt injection safeguard, causing the subagent to Fail. Proceed with targeted semantic locators (`find role button`) instead of full snapshots in hostile/large pages.
